# Contributing

1. Install `pnpm` globally (ignore if already installed).

   ```bash
   npm i pnpm -g
   ```

2. Install dependencies.

   ```bash
   pnpm install
   ```

3. Local development (Requires the corresponding browser to be installed).

   ```bash
   pnpm dev
   ```

4. Build

   ```bash
   pnpm build
   ```
